# result-type

[![npm version](https://img.shields.io/npm/v/result-type.svg)](https://www.npmjs.com/package/result-type)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Rust-style Result type for error handling without exceptions

## Installation

```bash
npm install result-type
```

## Functions

- `ok` — Create a successful Result
- `err` — Create a failed Result
- `tryResult` — Wrap a function call into a Result
- `tryAsyncResult` — Wrap an async function call into a Result
